## 📖 Git Convention

- **Start:** Start New Project
- **Feat:** 새로운 기능 추가
- **Fix:** 버그 수정
- **Refactor:** 코드 리팩토링
- **Settings:** Changing configuration files
- **Comment:** 필요한 주석 추가 및 변경
- **Dependency/Plugin:** Add a dependency/plugin
- **Docs:** 문서 수정
- **Rename:** 파일 혹은 폴더명을 수정하거나 옮기는 작업만인 경우# Temporary commit for PR
